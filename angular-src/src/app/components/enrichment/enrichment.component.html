Competitive Programming

This course will help students improve their skills in competitive programming, specifically USACO, as well as their problem solving and coding skills. The topics covered in this course are common interview questions in the field of Computer Science. Tentative topics include:

  • Prefix Sums
  • Sorting Algorithms
  • Graph Theory
  • Tree Traversal Algorithms
  • Greedy Algorithms
  • Hashing
  • Dynamic Programming
  • Computational Geometry

Java Greenfoot Summer Course

Instructor: Martin Ly

This course is suitable for entry level students who are interested in learning Java programming and basic game development. Prior experience is not required. The course will introduce the object-oriented Java programming language from the bottom up. Students will learn about good programming practices.
In this course, students will:

  • Learn Java syntax and their applications such as integrating input/output, calculations, logic/looping flow and control, object-oriented programming, exception handling, multi-threading, etc.
  • How to build a fully developed application or program using Java based on a simple idea or concept
  • Follow step by step game examples and exercises, then applying these fundamental concepts to create more complicated applications
  • Learn how to use a graphics editor to design simple 2D images
  • Intermediate students will be encouraged to use Netbean and begin learning about topics covered in AP Computer Science

Android App Development Summer Course

Children of all ages can use our easy-to-learn online development tool to develop an Android application We utilize user-friendly visual block programming, making this a great course for those without prior programming experience. Millions of applications and games can be built utilizing the built in abilities of the Android phone including the GPS tracking system, connection to the Web, gyro/magneto sensors, texting capabilities, email sending/receiving, voice recognition, clock, etc.
In this course, students will:

  • Learn all the functions and elements of blocks
  • Learn programming logic control and flow
  • How to create a user interface and set its layout
  • Learn about program animations and graphics, sensors and storage
  • Follow simple steps to create basic applications and games
  • Turn ideas into live applications
  • Learn in small classes of 4 to 5 students
  • Most suitable for students in grades 4 to 7
  • Students will need to bring their own laptop, and have access to a Android phone/tablet that runs on the Android OS
