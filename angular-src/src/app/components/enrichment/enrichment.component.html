Competitive Programming

This course will help students improve their skills in competitive programming, specifically USACO, as well as their problem solving and coding skills. The topics covered in this course are common interview questions in the field of Computer Science. Tentative topics include:

  • Prefix Sums
  • Sorting Algorithms
  • Graph Theory
  • Tree Traversal Algorithms
  • Greedy Algorithms
  • Hashing
  • Dynamic Programming
  • Computational Geometry

Introduction to Programming

This 10 week course is suitable for entry level students who are interested in learning Java programming and basic graphics. Prior experience is not required. The course will introduce the object-oriented Java programming language from the bottom up. In this course, students will:

  • Learn about data types, loops, and other fundamental concepts of programming
  • Learn Java syntax, good programming practices, and developing logic for programming
  • Learn about graphics and methods in Java
  • Learn about object-oriented programming
  • Intermediate students will learn more advanced concepts such as data structures
  • Take home their project files each week
